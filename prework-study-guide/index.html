HTML

  • HTML is the language that ceates the structure of the webpage.
  • An element is a type of content on a webpage.
  • Elements are also used to divide the content on the page and to describe the content to browsers and other developers.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Section elements are semantic elements that define a section in a document that contains thematic content.
  • When working with visible HTML elements, the content is contained between an opening tag and a closing tag.
  • These tags are most often represented by angle brackets with associated tag names contained inside.

Git

  • git allows you to make branches to work on code and then merge new code into the main branch.
  • First, you create a new branch, then you make changes and save them.
  • Then you commit the changes and push them to the github repository or "repo"
  • Once your code is ready for review, you create a pull request which allows others to check your work and approve it.
  • Once your pull request is approved, you can merge changes to the main branch, and delete the branch.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
